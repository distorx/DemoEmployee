# EmployeeCrud

ASP.NET MVC 5 web application for managing employees, companies, job positions, and service equipment with a Unit of Work pattern.

## What It Does

A CRUD application for employee management with AngularJS frontend modules. Manage employees, companies, job positions, and service equipment with user authentication via ASP.NET Identity.

## Features

- **Employee Management** — Create, edit, list employees with company and job position assignments
- **Company & Job Positions** — Manage organizational structure
- **Service Equipment** — Track equipment records
- **Authentication** — ASP.NET Identity with login, register, and account management
- **Unit of Work** — Repository pattern with IUnitOfWork for data access
- **Unit Tests** — Controller tests included

## Tech Stack

- **Framework:** ASP.NET MVC 5
- **Frontend:** AngularJS, Bootstrap
- **ORM:** Entity Framework 6 (Code First)
- **Database:** SQL Server (LocalDB)
- **Auth:** ASP.NET Identity
- **Language:** C#
